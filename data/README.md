# Data Directory

Large data files are excluded from version control via .gitignore.

## Structure

- `raw/` — Downloaded datasets (IEEE-CIS, PaySim, Credit Card)
- `processed/` — Preprocessed/feature-engineered datasets
- `synthetic/` — Generated synthetic scenarios

## Downloading Datasets

Instructions for acquiring each dataset:

1. **IEEE-CIS Fraud Detection**: https://www.kaggle.com/c/ieee-fraud-detection/data
2. **PaySim**: https://github.com/EdgarLopezPhD/PaySim (or Kaggle)
3. **Credit Card Fraud**: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
